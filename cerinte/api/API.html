This page will present the SkiVI's Microservices API

Request Syntax: "METHOD" {path?params}

For our Microservices: {path} = http://localhost:3000/SkiVI/microservices/{microservice}/api/product/{file}.php

{microservice} is one of origamimicroservice, cprmicroservice, knottingmicroservice

{file} is one of createLesson, deleteLesson, getAllLessons, getCourseLessons, getSingleLesson, updateLesson

Example of API calls

Using any other request method than that specified will result in an error message, simmilar to the "Getting all lessons available" failure response

Getting all lessons available

Uses: getAllLessons with no parameters

Request method: "GET"
